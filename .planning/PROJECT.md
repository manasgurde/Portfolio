# Cinematic Scroll Story Portfolio

A highly immersive, modern, responsive, and performance-optimized portfolio website that uses scroll-driven storytelling, cinematic transitions, motion graphics, and 3D elements. The goal is to create an experience that feels like watching an interactive movie, reflecting the premium design style of Awwwards Site of the Day winners.
