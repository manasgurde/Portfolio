# Stack Research

## Recommended Stack

| Layer | Technology | Version | Rationale | Confidence |
|-------|------------|---------|-----------|------------|
| Frontend | Next.js | 15+ | Industry standard, App Router performance. | High |
| UI/Styles | Tailwind CSS | 3.4+ | Utility-first, fast styling without separate CSS. | High |
| Animation | GSAP + Framer Motion | Latest | Essential for cinematic scroll animations. | High |
| 3D | Three.js + R3F | Latest | Immersive background elements. | High |
| Backend | Next.js API Routes | - | Sufficient for simple form submissions. | High |
| Database | Supabase (PostgreSQL) | Latest | Easy integration for storing contact messages or dynamic content. | High |
| Email | Resend | Latest | Developer-friendly email delivery for contact form. | High |

## Do Not Use
- Traditional CMS (WordPress) - overhead is too high, less control over complex animations.
- Complex state managers (Redux) - overkill for a portfolio.
