# Roadmap

## Phase 1: Foundation & Base Intro
**Goal:** Setup Next.js, basic layout, and the 3D hero environment
**Success Criteria:**
1. Next.js 15+ is running with Tailwind CSS
2. Three.js canvas renders floating particles
3. "Hello" -> "I'm [Name]" cinematic typography works
**Requirements:** FND-01, FND-02, FND-03, INT-01, INT-02, INT-03, INT-04

## Phase 2: Storytelling & Skills
**Goal:** Implement scroll-linked storytelling chapters and floating skill cards
**Success Criteria:**
1. Scroll reveals story chapters smoothly with parallax
2. Skill cards float and react to hover
**Requirements:** STR-01, STR-02, STR-03, SKL-01

## Phase 3: Projects Showcase
**Goal:** Build cinematic full-screen project reveals and details
**Success Criteria:**
1. Projects appear with cinematic full-screen reveal
2. Camera pan/zoom effects function correctly on scroll
**Requirements:** PRJ-01, PRJ-02, PRJ-03

## Phase 4: Journey, Achievements, Testimonials
**Goal:** Implement animated timeline, achievements, and testimonial carousel
**Success Criteria:**
1. Timeline builds itself on scroll
2. Achievements fade in smoothly
3. Testimonial carousel is navigable
**Requirements:** JRN-01, ACH-01, TST-01, PRO-01, PER-01

## Phase 5: Contact & Backend Integration
**Goal:** Build contact scene and wire up Supabase/Resend
**Success Criteria:**
1. Contact form submits successfully
2. Form is stored in DB and email is sent
**Requirements:** CON-01, CON-02, BKN-01, BKN-02
