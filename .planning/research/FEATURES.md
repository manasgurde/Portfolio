# Features Research

## Table Stakes (Must Have)
- Responsive design across all devices (Desktop, Mobile, Tablet)
- High performance (Lighthouse 95+)
- Accessibility (ARIA, Semantic HTML)
- Contact form with email delivery
- Basic SEO (Metadata, Open Graph)

## Differentiators
- Cinematic scroll-driven storytelling
- 3D elements and particle backgrounds
- GSAP scroll-linked animations and parallax
- Interactive floating skill cards
- Animated timeline

## Anti-features (Do Not Build)
- Multi-page navigation (breaks the single cinematic journey)
- User authentication/login (unnecessary for a portfolio)
- Commenting system
