# Phase 3 Discussion Log: Projects Showcase

*This log captures the interaction that led to the decisions in `03-CONTEXT.md`.*

## Discussed Areas

### 1. Project Layout Style
**Options Presented:**
- Horizontal scrolling carousel (User scrolls down, but projects slide in horizontally)
- Vertical full-screen sections (Each project takes up 100vh and snaps into place)
- Expanding grid (Projects appear as a grid, clicking one expands it full-screen)

**Selection:** Horizontal scrolling carousel (User scrolls down, but projects slide in horizontally).

### 2. Media Type
**Options Presented:**
- High-res Images / Screenshots
- Video demos or GIFs
- Interactive 3D device mockups (e.g. laptop/phone renders inside the canvas)

**Selection:** High-res Images / Screenshots.

### 3. Background Behavior
**Options Presented:**
- Camera zooms in deeply, pushing particles out of the way for clarity
- Camera orbits slowly around a central point, keeping particles as a subtle backdrop
- Camera freezes entirely to ensure the user focuses purely on the project details

**Selection:** Camera orbits slowly around a central point, keeping particles as a subtle backdrop.
